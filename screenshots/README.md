# Screenshots

This directory contains screenshots of the Invoice Pro landing page.

## How to Add Screenshots

1. Take screenshots of your landing page sections
2. Save them with the following names:
   - `hero-section.png` - Hero section with invoice preview
   - `features-section.png` - Features showcase
   - `how-it-works.png` - How It Works section
   - `why-choose.png` - Why Choose Invoice Pro section
   - `cta-section.png` - Call to Action section
   - `full-landing-page.png` - Complete landing page view

3. Recommended screenshot dimensions:
   - Section screenshots: 1920x1080 or 1440x900
   - Full page: Use browser screenshot extension for full page capture

4. After adding screenshots, they will automatically appear in the main README.md

## Tools for Screenshots

- **Browser Extensions**: 
  - Full Page Screen Capture (Chrome/Edge)
  - Fireshot
  - Awesome Screenshot
  
- **Built-in Tools**:
  - Windows: Snipping Tool or Win + Shift + S
  - Mac: Cmd + Shift + 4
  - Linux: Various screenshot tools
